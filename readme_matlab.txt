Guideline to run Matlab script
PointSourceGUI_v1_0.m
=====================================

Requires a running Matlab installation
-------------------

STEP 1) Start Matlab

STEP 2) Open PointSourceGUI_v1_0.m in Matlab Editor
Make sure that the files
 PointSourceGUI_v1_0.m 
 PointSourceGUI_v1_0.fig
 PointSourceClass.m
are in your current directory

STEP 3) Run PointSourceGUI_v1_0.m from Matlab Editor 
(click "Run" in the Editor Ribbon, or hit F5-key) 

--------------------------

Tested with Matlab 2019a (Win10)
